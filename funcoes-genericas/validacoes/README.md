# Validações

Pasta para funções genéricas de validação no Power Query M.

## Exemplos de uso futuro

- validar campos obrigatórios;
- validar tipos de dados;
- validar datas de fechamento mensal;
- validar chaves de conciliação;
- identificar nulos, vazios e inconsistências.

## Status

Pasta estrutural criada para organização futura de funções.
