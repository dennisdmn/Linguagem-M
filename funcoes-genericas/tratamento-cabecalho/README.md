# Tratamento de cabeçalho

Pasta para funções genéricas de padronização de nomes de colunas no Power Query M.

## Exemplos de uso futuro

- converter nomes de colunas para `snake_case`;
- remover acentos;
- remover caracteres especiais;
- padronizar espaços, quebras de linha e pontuação;
- criar cabeçalhos compatíveis com modelos analíticos.

## Status

Pasta estrutural criada para organização futura de funções.
