# Funções genéricas

Esta pasta reúne funções Power Query M reutilizáveis em diferentes projetos.

## Estrutura sugerida

- `importacao/`: funções para leitura de arquivos, pastas, SharePoint, Excel, CSV e fontes similares.
- `tratamento-cabecalho/`: funções para padronização de nomes de colunas, como snake_case/cabeçalho em cobrinha.
- `validacoes/`: funções de validação de campos, tipos, datas, chaves e regras de negócio.

## Uso por IA

Ao pedir auxílio a uma IA, indique a pasta ou função de referência. Exemplo:

```text
Consulte o repositório dennisdmn/Linguagem-M e use os padrões em funcoes-genericas/tratamento-cabecalho.
```
