# Balancete

Pasta reservada para scripts Power Query M relacionados a bases de balancete, razão, saldos contábeis e análises de fechamento.

## Finalidade

Centralizar funções e consultas específicas para tratamento, padronização e validação de balancetes, incluindo contas contábeis, centros de lucro, empresas, períodos, saldos e movimentações.

## Uso por IA

Ao solicitar apoio a uma IA, indique este caminho como referência:

```text
Consulte o repositório dennisdmn/Linguagem-M e use os scripts em casos-especificos/cogna/balancete como referência para rotinas de balancete.
```
