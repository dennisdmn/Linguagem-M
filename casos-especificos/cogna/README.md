# Casos específicos — Cogna

Esta pasta reúne scripts Power Query M voltados a rotinas, layouts e processos específicos da Cogna.

## Estrutura sugerida

- `fpg5/`: tratamento de extrações da rotina FPG5.
- `fagl/`: scripts relacionados a bases, extrações ou conciliações FAGL.
- `fpl9/`: scripts relacionados a consultas, bases ou validações FPL9.
- `comissoes/`: scripts voltados a análises de comissão, repasses e validações financeiras.

## Uso por IA

Ao solicitar ajuda a uma IA, informe o caminho da pasta ou do arquivo desejado. Exemplo:

```text
Consulte o repositório dennisdmn/Linguagem-M e use como referência os scripts em casos-especificos/cogna/fpg5.
```
