# FPT8

Pasta reservada para scripts Power Query M relacionados a bases, extrações, tratamentos e validações da rotina FPT8.

## Finalidade

Centralizar funções e consultas específicas para trabalhar com dados oriundos da FPT8, incluindo preparação de layout, padronização de campos, validações e cruzamentos com outras bases financeiras ou contábeis.

## Uso por IA

Ao solicitar apoio a uma IA, indique este caminho como referência:

```text
Consulte o repositório dennisdmn/Linguagem-M e use os scripts em casos-especificos/cogna/fpt8 como referência para rotinas FPT8.
```
